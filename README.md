🌸 Amara – Maternal Companion App

Amara is a smart maternal care companion designed to support women through pregnancy and early motherhood. It provides health tracking, personalized guidance, reminders, and educational resources to ensure a safe and informed journey for both mother and baby.

📱 Features
🤰 Pregnancy Tracking
Week-by-week fetal development updates
Trimester-based health insights
🩺 Health Monitoring
Track weight, blood pressure, and symptoms
Record doctor visits and reports
🔔 Smart Reminders
Medication alerts
Appointment scheduling notifications
📚 Educational Content
Nutrition tips
Exercise guidance
Do’s and Don’ts during pregnancy
🧠 AI-based Assistance (Optional)
Chat-based guidance for common queries
Personalized suggestions
🍼 Postnatal Care
Baby growth tracking
Vaccination reminders
🚀 Tech Stack
Frontend: HTML, CSS, JavaScript / React (if used)
Backend: Firebase / Node.js
Database: Firebase Firestore / Realtime DB
Hosting: Vercel / Firebase Hosting
🛠️ Installation & Setup
Clone the repository:
git clone https://github.com/mgayathrimonika-a11y/amara-maternal-companion-app.git
Navigate to the project folder:
cd amara-maternal-app
Install dependencies:
npm install
Run the app:
npm start
